# gulp-scss-boilerplate
Este boilerplate foi criado para um início rápido no desenvolvimento de sites.

O Boilerplate contém Gulp como task runner, preprocessors (SCSS).

Existem algumas tarefas no gulpfile.js
- compilação de pré-processadores em CSS;
- recarregamento automático de página (browsersync)
- Compactação de arquivos CSS
- task de criar projeto na pasta 'dist' antes da produção

Inicie um projeto deste modo:
- git clone https://github.com/indiorlei/gulp-scss-boilerplate
- cd gulp-scss-boilerplate;
- yarn install
- gulp watch

Para build do projeto (/dist)
- gulp build
